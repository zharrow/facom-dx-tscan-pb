## Utilisation prévue

Votre analyseur de disques de frein et de pneus a été conçu pour des interventions professionnelles.

C'est un instrument petit et pratique qui permet de détecter l'usure des disques de frein sur les véhicules sans avoir à démonter les roues. Il est également capable de mesurer la profondeur de la bande de roulement des pneus.

L'outil projette un faisceau laser sur le disque de frein ou sur le pneu ; l'image créée est détectée par une micro-caméra. Les données obtenues sont envoyées à l'afficheur sur lequel est installé le logiciel spécifique qui permet de les traiter. Le résultat est une présentation graphique du disque ou du pneu associée à l'état d'usure.

Pour obtenir plus d'informations, consultez le site Internet [www.facom.com](https://www.facom.com).

> **N'UTILISEZ PAS** le laser dans un environnement humide ou en présence de liquides ou de gaz inflammables.

> Ces instruments laser sont des outils électriques professionnels. **NE LAISSEZ PAS** les enfants manipuler l'outil. La surveillance des utilisateurs inexpérimentés est nécessaire.

- **Jeunes enfants et personnes handicapées.** Cet appareil n'est pas prévu pour être utilisé par de jeunes enfants ou des personnes handicapées sans surveillance.
- Ce produit n'est pas prévu pour être utilisé par des personnes (y compris les enfants) dont les capacités physiques, sensorielles ou mentales sont réduites, ou qui manquent d'expérience, de connaissances ou d'aptitudes, à moins d'être surveillées par une personne responsable de leur sécurité. Ne laissez jamais les enfants seuls avec ce produit.

### Menu Configuration

#### Paramètres

À partir du menu *Paramètres*, l'utilisateur peut configurer :

- Le port série COM (voir *Configuration Port COM*).
- Le nombre d'acquisitions pour le test des disques de frein (voir *Nombre d'acquisitions*).
- Le nombre d'acquisitions pour le test des pneus.
- La roue par laquelle la prise de mesure commence.
- Le sens de la prise de mesure sur la roue (horaire ou antihoraire).
- La limite d'usure du disque (mm).
- La limite de profondeur de la bande de roulement (mm).
- La langue utilisateur du logiciel.
- La résolution de l'écran.

#### Configuration Port COM

1. Cliquez sur *Paramètres*.
2. Cliquez sur *Rechercher*. Le logiciel affiche les analyseurs trouvés pendant le scan (type, numéro de série, port COM, état d'activation).
   - Pour **activer** un analyseur, déplacez le curseur vers la droite.
   - Pour **désactiver** un analyseur, déplacez le curseur vers la gauche.
3. Cliquez sur *Sauvegarder*.

#### Nombre d'acquisitions

1. Cliquez sur *Paramètres*.
2. Sélectionnez une valeur entre 1 et 5 dans les menus *Nombre d'acquisitions* (disques de frein et pneus).
3. Cliquez sur *Sauvegarder*.

> Le résultat calculé est la moyenne arithmétique des valeurs obtenues. Le logiciel affiche chaque acquisition dans un graphique avec des couleurs différentes.

#### Configuration Atelier

1. Cliquez sur *Données Atelier*.
2. Saisissez les informations requises dans les champs spécifiques.
3. Cliquez sur *Sauvegarder*.

### Analyse de l'usure des disques de frein (Fig. A, E)

> **AVERTISSEMENT :** n'allumez pas le laser avant d'avoir correctement positionné l'analyseur pour la prise des mesures.

Pour un résultat optimal, positionnez l'analyseur comme illustré à la figure E.

> **RECOMMANDATION :** prenez plusieurs mesures si le disque est très corrodé.

> **RECOMMANDATION :** assurez-vous que le système de freinage du véhicule fonctionne tel que spécifié par le fabricant.

**Préparation**

1. Retirez l'adaptateur pour l'analyse de l'usure des pneus (6) ou l'adaptateur pour la vérification du système de mesure (7) (voir *Adaptateur*).
2. Allumez l'analyseur en appuyant sur le bouton multifonction (1).
3. Lancez le logiciel SCANDIAG et cliquez sur *Disque de frein*.

**Acquisition des données**

1. Positionnez l'analyseur comme illustré à la figure E, en vous assurant que l'aimant (5) est parfaitement collé au disque de frein.
   > **REMARQUE :** les rayons de la roue ne doivent pas se trouver entre le faisceau laser et le disque de frein.
2. Enfoncez le bouton multifonction (1) pour activer le laser.
3. Dirigez l'analyseur de sorte que le faisceau laser soit le plus possible aligné avec le centre du disque de frein.
4. Cliquez sur le bouton multifonction (1) pour activer la micro-caméra. Un signal sonore indique que la mesure a été prise.
5. Un second signal sonore confirme l'envoi des données au logiciel. Les données sont affichées sur l'afficheur/PC.
   - Répétez le procédé pour chaque roue.

**Résultats**

Les données acquises sont affichées de façon graphique, les disques de frein étant colorés.

- Pour sauvegarder un rapport, cliquez sur *Imprimante*.
  > **REMARQUE :** si vous cliquez sur *Retour*, vous supprimez le rapport.
- Pour supprimer un rapport, cliquez sur *Effacer*.

| Couleur | Signification (disque de frein) |
|---|---|
| Noir | Usure non détectée ou en attente de prise de mesure. |
| Vert | L'usure est dans la limite définie au menu *Paramètres*. |
| Jaune | L'usure est proche de la limite définie. |
| Rouge | L'usure ne respecte pas la limite : remplacer le disque de frein. |

Pour imprimer le rapport, cliquez sur *Imprimer* puis sur *Aperçu avant impression*.

## Maintenance

Votre analyseur a été conçu pour fonctionner longtemps avec un minimum de maintenance. Son fonctionnement continu et satisfaisant dépend de son bon entretien et de son nettoyage régulier.

- Respectez scrupuleusement les instructions de ce manuel.
- Veillez à ce que le produit reste propre.
- Inspectez régulièrement les raccords électriques. N'alimentez l'outil qu'avec le chargeur fourni.
- Remplacez immédiatement tout câble endommagé.
- N'utilisez que des pièces détachées d'origine ou approuvées par le fabricant (voir *Contenu de l'emballage* pour les numéros d'articles).
- Contactez votre revendeur pour toute opération de maintenance sortant de l'ordinaire.

### Nettoyage

> **AVERTISSEMENT :** n'utilisez jamais de solvant ou autre produit chimique décapant pour nettoyer les parties non métalliques de l'outil — ces produits pourraient fragiliser la matière. Utilisez un chiffon humidifié avec de l'eau et un savon doux. Ne laissez jamais aucun liquide pénétrer à l'intérieur de l'outil et n'immergez jamais aucune pièce dans aucun liquide.

| | Quelque chose fait écran entre l'analyseur et/ou l'afficheur/ordinateur. | Supprimez ce qui fait écran. |

| Détection incorrecte du disque de frein ou du profil du pneu. | L'adaptateur est en place pendant l'analyse du disque de frein. | Retirez l'adaptateur. |

| | Le faisceau laser n'est pas bien aligné avec le centre du disque. | Redirigez le faisceau laser. |
| | L'aimant n'est pas correctement fixé (angle d'incidence incorrect). | Fixez correctement l'aimant sur le disque de frein. |

| | L'outil a besoin d'être calibré. | Voir *Contrôles périodiques de l'outil*. |
| | L'aimant n'est pas correctement inséré dans l'adaptateur. | Contrôlez et retirez tout corps étranger ou saleté sur l'aimant et/ou l'adaptateur. |
| | Des corps étrangers (ex. graviers) sont coincés dans les rainures du pneu. | Retirez-les ou repositionnez l'outil dans une zone sans corps étrangers. |
| | L'adaptateur n'est pas à plat sur la bande de roulement. | Repositionnez l'analyseur. |
| | L'analyseur n'est pas positionné sur une rainure centrale du pneu. | Repositionnez l'analyseur. |
| | Le faisceau laser n'est pas assez perpendiculaire à la rainure. | Redirigez le faisceau laser. |
| | Le faisceau laser dépasse les témoins d'usure de la bande de roulement. | Repositionnez l'analyseur. |
